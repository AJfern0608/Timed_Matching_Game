Memory Match Game Documentation

Complete guide to running and deploying the game on Replit

Game Overview

Memory Match is a card matching game with multiple difficulty levels. Test your memory by finding matching pairs of cards before time runs out. The game features:

  • 5 progressive difficulty levels, each with more cards and challenges
  • Score tracking with local high score saving
  • Lives system and time limits to increase challenge
  • Responsive design for different screen sizes

How to Play

  1. Click on a card to flip it over and reveal the symbol
  2. Try to find the matching card with the same symbol
  3. If you find a match, the cards stay flipped and you earn points
  4. If not, both cards are flipped back over
  5. Match all cards before time runs out to advance to the next level
  6. If time runs out, you lose a life
  7. Complete all 5 levels to win the game!
Tip: Try to memorize the positions of cards you've already seen to make matches more efficiently!

Project Structure

The game is built using React with TypeScript and uses the following main components:

Key Files

  • client/src/MatchingGame.tsx - Main game component with all game logic
  • client/src/MatchingGame.css - All game styling
  • client/src/App.tsx - Root application component
  • server/index.ts - Express server that serves the application

Dependencies

The game relies on the following technologies:

  • React - UI library
  • TypeScript - Type-safe JavaScript
  • Vite - Fast build tool and development server
  • Express - Web server

Deploying to Replit

Follow these steps to deploy the Memory Match game to Replit:

Option 1: Clone and Deploy

  1. Create a new Node.js project on Replit
  2. Delete any starter files Replit creates
  3. Import all files from this project to your Replit project
  4. In the Replit shell, run npm install to install dependencies
  5. Run npm run dev to start the game

Option 2: Manual Setup

  1. Create a new Node.js project on Replit
  2. Copy the package.json file first
  3. Run npm install to install dependencies
  4. Copy all other project files
  5. Run npm run dev to start the game
Important: Make sure your Replit project is set to run npm run dev as its start command.
