Memory Match Game

A fun card matching game with multiple difficulty levels!

How to Play

  1. Click on a card to flip it over.
  2. Try to find matching pairs of cards.
  3. Complete all levels before running out of lives.
  4. Advance through 5 progressively difficult levels.
  5. Try to achieve the highest score possible!

How to Run on Replit

  1. Copy the entire project to your Replit.
  2. In the Replit shell, run npm install to install dependencies.
  3. Run the game with npm run dev.
  4. Click the "Run" button in Replit to start the server.
  5. Your game will be accessible at the URL provided by Replit.
